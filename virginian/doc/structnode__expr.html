Detailed Description

Defines an expression node of the AST.

Expressions are how values are represented and combined through mathematical operators. A value can be either a constant or drawn from a data record. Expression nodes can represent a constant value, a value drawn from a data record, or an operation used to combine two sub expressions, thus a tree of expressions can be built
