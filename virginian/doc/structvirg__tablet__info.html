Detailed Description

Information associated with a tablet on disk.

This struct is used in a dynamically allocated array to store the locations of tablets on disk. To find a tablet on disk, these will be iterated over. Note that though a tablet is in disk it may also be in memory, possibly with changes that have not been flushed to disk
